YourWeb.Expert Styleguide

Button

Standard button.

States

  • :hover - highlights when hovered
  • :disabled - dims the button when disabled.

Example

<button class="o-button">click me</button>

Input

Standard text input

States

  • :disabled, --disabled - Dims the input when disabled
  • .input--block - Makes the input full-width

Example

 <input class="o-form__input" id="input" placeholder="placeholder text" type="text"></input>
 <input class="o-form__input" disabled id="input" placeholder="placeholder text" type="text"></input>
 <input class="o-form__input o-form__input--block" id="input" placeholder="placeholder text" type="text"></input>

Label

Standard label for form inputs

States

Example

<label class="o-form__label" for="input">Form field:</label>

Image

Standard image

States

  • --centred - centres the block-level image
  • --full-width - image becomes full width of the parent container

Example

 <img class="o-image" src="http://placehold.it/200x200" alt="Alt text"></img>
 <img class="o-image o-image--centred" src="http://placehold.it/200x200" alt="Alt text"></img>
 <img class="o-image o-image--full-width" src="http://placehold.it/200x200" alt="Alt text"></img>
